# Basket Points
Marcador de puntos anotados en un encuentro de basketball modificando un Layout para su proporcionalidad en pantalla.
## Motivacion
Uso de metricas de diseño de UI por parte de Udacity.
## Instalacion
Android API 21 (Lollypop) y posterior.
